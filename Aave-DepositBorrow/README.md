# Aave Deposit & Borrow Mini-dApp

## Overview
This project demonstrates a mini DeFi dApp that interacts with Aave V3. Users will be able to:
- Deposit ERC20 tokens (e.g., DAI)
- Borrow ERC20 tokens (e.g., USDC)
- Track balances and interest rates

## Tech Stack
- Solidity
- Hardhat
- OpenZeppelin Contracts
- Goerli Testnet
- MetaMask

## Features
- Deposit tokens into Aave lending pool
- Borrow tokens
- Repay borrowed tokens
- Unit tests (to be added)
